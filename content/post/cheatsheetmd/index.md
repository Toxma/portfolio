## Header

    # h1
    ## h2
    ### h3

## Bold

    **bold text**

## Italic

    *italic text*

## Highlight

    ==highlighted text==

## Strikethrough

    ~~strikethrough text~~

## Block quote

    > quote

## Ordered list

    1. Item 1
    1. Item 1.1
    2. Item 2
    3. Item 3 

## Unordered list

   - Item 1
     - Item 1.1
   - Item 2
   - Item 3 

## Inline Code

    `code`

## Code Block

    ```<language>
    
    ```
